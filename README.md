# Тестовые задания от компании YANDEX на Backend направление, осень-зима 2025.
Тестовые задания от компании YANDEX на Backend направление, осень-зима 2025.

**Используемые языки:**
* Python

**Содержание работ:**
1) ***Сокровища островов пирата Дино***. Имеется *n* островов с *m* тунелями между ними, на каждом острове есть сокровища ценностью $v_i$. Необходимо **собрать как можно больше сокровищ при условии, что Дино не может посещать уже пройденные острова** (стартует он с острова №1, а завершить путешествие может где угодно). 
2) ***Отпуск Дино и затопление островов***. Имеется карта *n* x *m* с высотами островов. Каждую минуту вода поднимается на 1 метр. Определить, **в какой момент времени будет затоплена каждая ячейка** этой карты.
3) ***Обогащение контекста***. **Реализовать алгоритм объединения запросов** (контекста), если в них совпадает хотя бы одно слово. Вывести количество различных по всем словам контекстов (с учётом объединения контекстов в один), а также длину самого большого из них.
4) ***Минимальное время разрушения платформ***. Динозавр бежит по цепочке из *N* платформ. Каждая платформа имеет высоту $h_i$. Платформа на позиции *i* разрушается, если существует платформа *j* (где $j > i$), удовлетворяющая следующим условиям: *i* и *j* имеют одинаковую чётность (т.е. *i mod 2 ≡ j mod 2*), $h[j] > h[i]$. Время разрушения платформы *i* - это минимальное значение *j−i* среди всех подходящих *j*. Если таких платформ *j* нет, время разрушения считается равным *−1*. **Определите время разрушения для каждой платформы**.
