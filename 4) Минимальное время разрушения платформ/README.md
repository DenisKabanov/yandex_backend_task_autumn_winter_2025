# Минимальное время разрушения платформ


## Задание:
Динозавр бежит по цепочке из *N* платформ, пронумерованных от *0* до *N−1*. Каждая платформа *i* имеет высоту $h_i$. Платформа *i* разрушается, если существует платформа *j* (где $j > i$), удовлетворяющая следующим условиям:
1) *i* и *j* имеют одинаковую чётность (т.е. *i mod 2 ≡ j mod 2*);
2) $h[j] > h[i]$.

**Время разрушения платформы *i* - это минимальное значение *j−i* среди всех подходящих *j***. Если таких платформ *j* нет, время разрушения считается равным *−1*.<br> 
**Определите время разрушения для каждой платформы**.


## Формат ввода (стандартный ввод или input.txt):
Первая строка содержит целое число ***N*** (*1 ≤ N ≤ 10*) - количество платформ.<br> 
Вторая строка содержит *N* целых чисел $h_0, h_1,...,h_{N-1}$ ($1 ≤ h_i ≤ 10^9$) - высоты платформ.


## Формат вывода (стандартный вывод или output.txt):
Выведите N целых чисел, где *i-е* число - минимальное время разрушения платформы *i* или *−1*, если разрушение невозможно.


## Примеры:
1) 
    **Ввод:**
    ```
    6
    5 3 8 1 7 9
    ```

    **Вывод:**
    ```
    2 4 -1 2 -1 -1
    ```
    *Примечание.* По всем числам:
    * для числа 5 (на чётной позиции i=0) ближайшее число, что больше и тоже на нечётной позиции — это 8, что стоит на позиции j=2 → $j-i = 2-0 = 2$;
    * для числа 3 (на нечётной позиции i=1) ближайшее число, что больше и тоже на нечётной позиции — это 9, что стоит на позиции j=5 → $j-i = 5-1 = 4$;
    * для числа 8 (на чётной позиции i=2) нет такого числа, что стояло бы праве на чётной позиции и было бы больше (9 стоит на нечётной);
    * для числа 1 (на нечётной позиции i=3) ближайшее число, что больше и тоже на нечётной позиции — это 9, что стоит на позиции j=5 → $j-i = 5-3 = 2$;
    * для числа 7 (на чётной позиции i=4) нет такого числа, что стояло бы праве на чётной позиции и было бы больше (9 стоит на нечётной);
    * для числа 7 (на нечётной позиции i=5) нет такого числа, что стояло бы праве на нечётной позиции и было бы больше (это и так последнее число в списке);
